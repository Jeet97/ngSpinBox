Ng-Spin-box

Spinbox Directive for Angular 2+ which provides cross-browser functionality of <input type='number'> with in-built supoort for Precision.

Usage

In app.module.ts or custom-module.module.ts 

    
      import { NgSpinBoxModule } from 'ng-spin-box';
and 
    
        imports: [
          NgSpinBoxModule,
        ]

In your Template, use as 

    
      <div>
        <input type='tel' ngSpinBox step='0.001' min='0.001' decimal='3'>
      </div>

Classes to customise spin-box

Wrapper Element: 'ng-spin-box-wrapper'
Increment Button: 'ng-spin-box-btn-up'
Decrement Button: 'ng-spin-box-btn-down'
